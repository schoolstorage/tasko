# Takso
```shell

https://serokell.io/blog/introduction-to-phoenix

# create new project
mix phx.new takso

# create database ( config/dev.exs )
mix ecto.create

# start dev server ( default: http://localhost:4000 )
mix phx.server

# create migration
mix ecto.gen.migration user

# run migrations
mix ecto.migrate
```
